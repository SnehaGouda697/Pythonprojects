# 🔐 Python Login System

A simple command-line login system built using Python. This project verifies a user's username and password using a dictionary and demonstrates basic Python concepts.

## 🚀 Features

- Username authentication
- Password validation
- Dictionary-based login
- Nested `if` statements
- `match-case` statement
- Membership (`in`) operator
- Identity (`is`) operator

## 🛠️ Technologies

- Python 3.10+
- VS Code

## ▶️ How to Run

```bash
python LoginSystem.py
```

## 📌 Concepts Used

- Dictionary
- User Input
- Conditional Statements
- Nested `if`
- `match-case`
- Membership Operator (`in`)
- Identity Operator (`is`)

## 📖 Purpose

This project is created for learning Python basics and practicing login authentication using conditional statements and dictionaries.
